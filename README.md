# 🏗️ Fork do Site Brasriodrywall.com.br

Este repositório contém um fork do site **brasriodrywall.com.br**, uma empresa especializada em soluções de drywall no Rio de Janeiro.

## 📋 Sobre o Projeto

O Brasriodrywall é uma empresa consolidada no mercado de construção civil, focada em:
- Instalação e montagem de sistemas drywall
- Acabamentos em gesso acartonado
- Soluções para divisórias e forros
- Projetos residenciais e comerciais

## 🎯 Objetivo do Fork

Este fork foi criado para:
- **Manutenção independente** do código do site
- **Implementação de melhorias** e novas funcionalidades
- **Testes de desenvolvimento** sem afetar o site principal
- **Personalização** para necessidades específicas

## 🛠️ Como Executar

1. **Clone o repositório:**
   ```bash
   git clone [url-do-repositorio]
   ```

2. **Navegue até o diretório:**
   ```bash
   cd brasriodrywall-fork
   ```

3. **Abra o arquivo index.html** em seu navegador ou configure um servidor local
